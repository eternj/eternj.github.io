2026年拉斯维加斯自驾游（四）：国家公园

Posted on 2026-03-17 Edited on 2026-03-18

（This article is still in progress …)

这一篇博文来记述一下这次自驾游玩几个国家公园的经历。

拱门国家公园（Arches National Park）

拱门国家公园是位于犹他州Moab的一个国家公园，以其独特的地貌以及由红砂岩经过上百万年风化形成的各种各样的拱门闻名。

峡谷地国家公园（Canyonlands National Park）

峡谷地国家公园是位于犹他州Moab的一个国家公园，紧邻拱门国家公园，两个公园可以一起游玩。峡谷地里面有一个梅萨拱门（Mesa Arch），是观看日出的著名景点。

大峡谷国家公园（Grand Canyon National Park）

大峡谷又称科罗拉多大峡谷，是美国知名度最高的国家公园之一。大峡谷国家公园位于亚利桑那州，分为南缘和北缘，我们这次去的是南缘。

2026年拉斯维加斯自驾游（三）：羚羊谷及其他

Posted on 2026-03-17 Edited on 2026-03-18

这篇博文记录我们游玩的几个非国家公园但是很有特色的地方。

下羚羊谷（Lower Antelope Canyon）

羚羊谷是位于亚利桑那州佩吉市（Page）纳瓦霍原住民保护区内的世界著名狭缝型峡谷，以其独特的地貌和变幻莫测的的光影效文明。羚羊谷是柔软砂岩经过上百万年的暴洪和风蚀形成的。羚羊谷分为上羚羊谷和下羚羊谷。上羚羊谷上窄下宽，内部是A字形状，以夏季中午的光束最著名。上羚羊谷地面平坦，适合儿童和长者游玩。下羚羊谷需要沿金属楼梯向下深入到25米深的地底，地形上宽下窄，内部更像V字形状。下羚羊谷地形比上羚羊谷狭窄，变化更多，更具有探险感。需要注意的是上下羚羊谷的游览均需要跟随纳瓦霍族导游游览，必须提前几个月预定门票。我们在计划这次行程的时候经过研究决定去下羚羊谷，并提前两个月在网上预订，预定的网站是：Dixie’s Lower Antelope Canyon Tours。根据网上的推荐，我们预定的是上午11点15分的团，因为网上说11点前后是光线最好的时候。

下羚羊谷的游览由两家纳瓦霍族旅游公司承包，每天规定只能接待3000人（每一个公司每天只能接待1500人）。游客们被分到15个人一组的游览小组，每组由一位导游带领。我们的导游是一个叫艾萨雅的男孩，看上去像个大学生，一头卷发，声音温柔，整个人给人很清澈的感觉。据他自己介绍，他是Hopi族人，是所有导游中唯一一位非纳瓦霍族人。

在艾萨雅的带领下我们到了峡谷的入口处，在这里等待攀上梯子进入谷底。据导游介绍，在旺季的时候（夏季）游客通常需要等待超过一个小时才能进入谷底。想象一下在100华氏度高温下无遮挡暴晒等待一个小时的情景，我们很庆幸是三月份来参观的。进入谷底的梯子比较陡，有一段需要面对梯子向下攀爬。下面两张照片是分别从地面和谷底照的入口梯子。

羚羊谷里只允许拍照不允许录像，据说原因是微软和苹果购买了羚羊谷的录像版权。不管怎样，我们一下到谷底就被独特的地貌和奇幻的曲线迷住了。我们随着导游在峡谷里面穿行，一面欣赏着不同光影和颜色的组合，一面惊叹大自然的鬼斧神工，没想到竟然可以看到如此美景。

2026年拉斯维加斯自驾游（二）：生命不息，音乐不止

Posted on 2026-03-17 Edited on 2026-03-18

这次自驾游一个主要的目的就是去看赵传和齐秦在拉斯维加斯的演唱会，这篇博文就先来记录一下这两场演唱会。

赵传

认识赵传还要从我上高中时的某一个冬日说起。当年正放寒假，我顶着漫天的雪花去到家乡唯一的商场闲逛。那时候商场里还有柜台，待售的商品都被放置在柜台里的货架上面，人们购买商品时需要请柜台里的售货员帮忙拿取要买的东西。那时候的音乐歌曲还是以磁带形式传播发行的，而售卖磁带的柜台会有一个大录音机随机播放新到的歌带。就是在这么一个寒冷的冬日，让我撞到了一个高亢的声音唱着《噢！莎莉》。这本应是一首伤心的情歌，可是竟然让这个钢铁般的声音唱出了别样的味道。接着撞入耳膜的便是那首独一无二的《我很丑，可是我很温柔》。一样高亢的声音，把一个故事缓缓道来，让我看到了一个对音乐无限热爱执着追求的灵魂。我瞬间被这个叫赵传的丑男人吸引。

之后的好多年，我听了赵传无数的歌，从《我是一只小小鸟》，到《像我一样骄傲》，到《爱我就给我》，到《爱要怎么说出口》，到《给所有知道我名字的人》，到《男孩看见野玫瑰》，到《因为爱》，到《逆风飞行》，还有很多很多。他的音质独一无二，他的高音清脆有力，他的感情真挚热烈。

赵传演唱会

“我很丑，可是我很温柔。外表冷漠，内心狂热，那就是我~~~” 3月7日的演唱会是以这几句歌词开始的。当歌声响起时，赵传还没有出现在舞台上，可是我仿佛已回到了几十年前的那个冬日，看到那个站在商场柜台边的我，第一次听到这个声音时心灵感受到的碰撞。这一次，我真的可以见到赵传真人了！然后，他出现了，舞台灯光骤亮，赵传就那么站在舞台中间，戴一顶牛仔帽，穿着一件马甲，一条牛仔裤，就那么出现在我眼前。我好激动啊，我的偶像在眼前，一如想象中的丑，可是也一如想象中的铁骨柔情。我的眼睛湿润了，但是不妨碍我跟着唱那些听过无数遍的歌曲。

在演唱会上，赵传和我们分享了他的音乐历程，讲述了他的音乐故事，他成立的红十字乐队以及他和其他音乐人之间的经历。他唱了很多歌迷们耳熟能详的他自己的歌，也唱了一些其他音乐人（包括Beyond乐队和李宗盛）的歌向他们致敬。期间赵传还从舞台上下到观众席和歌迷们握手留影，现场气氛很是热烈。这是一个忘我的夜晚，这是一个欣赏音乐的夜晚，这是一个回忆青春的夜晚。谢谢你，赵传！

2026年拉斯维加斯自驾游（一）：总览

Posted on 2026-03-16 Edited on 2026-03-18

小的时候港台歌曲流行，我听多了就有了几个自己的偶像，其中最喜欢的包括齐秦、童安格、王杰、赵传、周华健等等。今年早些时候得知赵传和齐秦会相继到拉斯维加斯开演唱会，我激动得不得了。从来只是在唱片和电视里听他们唱歌，这次我要到现场去听偶像们唱歌。赵传和齐秦的演唱会一前一后隔开一个星期（赵传是3月7日，齐秦是3月14日），那就借这个机会干脆来个拉斯维加斯周边十日游。

这几篇博文记载了我们这次自驾游的经历。写作它们的目的主要是给自己留个纪念，以后想要回忆的时候可以再翻开来看看。同时也希望能给读到它们的网友一点借鉴，如果我们的游记和经历能够帮助到你们的旅行，哪怕是一点点，我们就很开心了。

行程安排

这次旅行我们计划先到拉斯维加斯听赵传的演唱会，然后在地图上顺时针跑一个大圈，先后游玩拱门国家公园（Arches National Park）、峡谷地国家公园（Canyonlands National Park）、马蹄湾（Horseshoe Bend）、羚羊谷（Antelope Canyon）、大峡谷国家公园（Grand Canyon National Park），最后再回到拉斯维加斯听齐秦的演唱会。根据 Google Map 的估算，这一趟开下来大概有2300英里（3700公里）。下面是我用 Google Map 做的大概行程安排（点击看大图）。

这次自驾游的原则是把行程放得比较宽松，不去追求游玩的国家公园的数目，而是能轻松地把每个地方都玩好，每天开车的时间尽量控制在4.5个小时以内并且尽量不要在天黑时开车。本着这个原则，我们把原本计划要去的锡安国家公园和布莱斯峡谷国家公园从行程上去掉了。最终的行程如下：

  • 3月6日从家出发开到Bakersfield，车程约4.5个小时。在Bakersfield住一晚上。
  • 3月7日从Bakersfield出发开往拉斯维加斯，车程约4个小时。晚上听赵传演唱会。在拉斯维加斯住一晚上。
  • 3月8日从拉斯维加斯出发开往Moab UT，车程月6.5个小时。傍晚到峡谷地国家公园看日落。在Moab住第一个晚上。
  • 3月9日白天游玩拱门国家公园，晚上在拱门国家公园观星。在Moab住第二个晚上。
  • 3月10日早上到著名的峡谷地国家公园里的Mesa Arch看日出。然后回酒店补觉。中午从Moab UT开往Page AZ，车程约4.5个小时。途经163号公路上的Forest Gump Point（这是电影《阿甘正传》里面一个经典画面的拍摄地点）。傍晚到马蹄湾看日落。在Page住一晚上。
  • 3月11日早上睡到自然醒，然后打卡网红点Shell加油站，接着游玩下羚羊谷。下午开往大峡谷国家公园，车程约2.5个小时。傍晚在Yavapai Point观看日落，晚上在Yavapai Point观星。在大峡谷住第一个晚上。
  • 3月12日早上在Mather Point看日出，下午坐公园里的红线接驳车游玩Hermit Road，傍晚在大峡谷边上的Rim Trail散步。在大峡谷住第二个晚上。
  • 3月13日早上从大峡谷开往拉斯维加斯，车程约4.5个小时。中间在Kingman AZ停留参观66号公路博物馆。晚上在拉斯维加斯大道上逛街。在拉斯维加斯住第一个晚上。
  • 3月14日白天在Las Vegas North Premium Outlets逛街购物，晚上看齐秦演唱会。在拉斯维加斯住第二个晚上。
  • 3月15日从拉斯维加斯开回家，车程约8.5个小时。

这个行程里3月8日和3月15日两天需要开车超过4.5个小时，这个是赶路需要，而且我们有两个人可以换手轮流开车，最后开下来觉得还可以接受，不是特别累。

阅读完陈浩基的《13.67》

Posted on 2026-02-23

今年开年许的愿望之一就是重新开始读书。目标很低：一个月读一本。一月份开始读 Ursula Le Cuin 的《No Time to Spare》，前面几章关于年长之后的人生思考以及她家猫猫的内容我读得还比较有兴趣。不过后面开始写她当作家时和读者的一些交流互动不是很吸引我，就暂时放弃了。

之后选择了香港作家陈浩基的《13.67》。这是一本中篇推理小说，全书有六个本格推理故事，作家用年代倒叙的手法，通过对关振铎和骆小明警察师徒破获的六个刑事大案的描写，描绘了1967年至2013年香港风起云涌的社会变迁（后来才明白书名是从这两个年代取的）。里面的故事构思巧妙，很多反转，首尾衔接，是不可多得的犯罪小说佳品。推荐给大家看。

How to Sideload Kodi to Apple TV Using atvloadly

Posted on 2026-02-15 Edited on 2026-02-23

Overview

The goal of this article is to share how I installed (sideloaded) Kodi to my Apple TV 4K (2nd Gen). Kodi is a media center and entertainment hub that I have been using since pre-Kodi era (when Kodi was XBMC). It has a beautiful interface and is easy to use. I mainly use Kodi to watch and record over-the-air (OTA) TV programs, as well as watch ripped blueray movies. Kodi has powerful add-ons that can expand and enhance its features and capabilities. It officially supports Windows, linux, and android. Unfortunately, Apple TV, which runs tvOS, does not have an official version of Kodi in App Store. This means we cannot download and install Kodi from App Store. Instead, we need to find some unofficial way to install or sideload it to Apple TVs.

atvloadly is a web service that supports sideloading apps on Apple TV. In this article, I’ll share how I setup an atvloadly docker and how I used atvloadly to sideload Kodi to my Apple TV. Hopefully this will help someone else who’s looking to do the same.

Workflow and Prerequisites

  1. Workflow

    • Setup an atvloadly docker.
    • Add an Apple account to atvloadly.
    • Pair the atvloadly web service with the Apple TV.
    • Sideload Kodi to the Apple TV.

  2. Prerequisites

    • A computer running linux. Unfortunately, this method does not work on Windows. It doesn’t need to be a full-fledged laptop or desktop. You can use a Raspberry Pi if you have one lying around. I used a single board computer called Rock64 running a stripped down version of Debian.
    • A burned Apple account. As noted on the atvloadly website, it is a good ideal to use a dedicated Apple ID or Apple account for sideloading. A free Apple account will work. I just created a new free Apple account for this purpose. You can create one on the Apple Developer Account webpage.
    • A phone for two-factor authorization (2FA). When creating a new Apple account for sideloading, you need a phone for 2FA. atvloadly needs to be authorized as a trusted device (it will be virtualized as a MacBook). When adding the Apple account to atvloadly, Apple will send a 2FA verification code to the registered phone number and you will need to enter the code and authorize it.
