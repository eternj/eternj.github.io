Unifi Network Setup

Posted on 2026-03-24 Edited on 2026-03-25

This is a simple note to myself regarding the network setup with Ubiquiti gears.

Network Topology

My home network topology is shown below:

My internet is provided by AT&T Uverse. The AT&T gateway (BGW210 Gateway) is connected to the WAN port (Port 5) of a Unifi cloud gateway fiber (UCG-Fiber). Then a Unifi 24-port switch (USW-24-POE) is connected to the first LAN port (Port 1) of the UCG-Fiber. Three other unifi equipment are connected to the USW-24-POE: a Unifi 8-port switch (USW-8-POE), a U6 Pro AP, and a U6 Lite AP.

Wireless clients are connected to the two APs while wired ones are connected to the USW-24-POE and the USW-8-POE.

Put BGW210 Gateway in Passthrough Mode

The first thing to do is to put AT&T BGW210 gateway in the passthrough mode. Follow the steps below:

  1. Log into the AT&T BGW210 gateway homepage and go to the “Firewall” tab on the top.

  2. Click on the “IP Passthrough” sub-tab. On this page, select “Passthrough” for Allocation Mode. Select “DHCPS-fixed” for Passthrough Mode. Then input your Unifi gateway’s MAC address below. It should look like this:

    Don’t forget to click on the “Save” button to save the settings.

  3. Disable Packet Filter, NAT Default Server, and all firewall rules.

  4. Disable WiFi on the BGW210 gateway since we are going to setup a Unifi wireless network.

Optimize Unifi WiFi Network

Steps to optimize the Unifi WiFi network:

  1. Using Unifi Design Center (UDC) to find the optimum placement for your APs. You can upload your home floor plan to UDC and experiment with different locations and power settings of your APs. The software will show you the WiFi coverage of your home based on your AP locations and power settings.
  2. 2.4GHz WiFi settings:
    • Use 20MHz channel width;
    • Use different channels for different APs to avoid interference;
    • Only select Channel 1, 6, or 11 for 2.4GHz.
  3. 5GHz WiFi settings:
    • Use 80MHz channel width;
    • Use different channels for different APs to avoid interference.
  4. Use WiFiMan to scan the RF environment to find optimum channels for your 2.4GHz and 5GHz network.
  5. Manually set the transmission power of the APs. The transmission power of 2.4GHz should be ~7dBm lower than that of 5GHz.

Set STP Priority

STP Priority needs to be set on the USW24 PoE and USW8 PoE. The STP priority can’t be the same for them. In my case, I set USW-24-POE to a STP priority of 8192 and USW-8-POE 16384.

Enable IPS/IDS

Enable IPS/IDS in Settings –> CyberSecure. As of this writing, there is only an option for IPS. IDS is missing from the settings.

Reduce BufferBloat

Steps to reduce bufferbloat:

  1. Go to Waveform to test your network’s bufferbloat.
  2. If the bufferbloat is high (you’ll get a rating lower than B in the first step), you need to turn on SQM in the UCG Fiber.
  3. Go to Settings –> Internet –> Select AT&T Internet and Check “Smart Queues”. Enter a download/upload rate that is 90%~95% of your ISP speed. Go back to Step 1 and test again. Some tuning is needed to get satisfied results.

Enable Flow Control

Enable flow control by going to Settings –> Internet and check “Flow Control”.

2026年拉斯维加斯自驾游（六）：大峡谷国家公园

Posted on 2026-03-21 Edited on 2026-03-22

大峡谷（Grand Canyon）也叫做科罗拉多大峡谷，位于美国西部的亚利桑那州西北角。大峡谷由科罗拉多河经过上百万年的冲刷形成，整个峡谷东西走向，全长277英里（446公里），最宽处18英里（29公里），平均深度1英里（1.6公里）。大峡谷在1979年被列为世界自然遗产，它是地球上最壮观的峡谷，是从太空中能看到的地貌之一。

向大峡谷国家公园进发

大峡谷分为南缘和北缘，其中南缘一年四季开放，配备有许多游客服务设施。而北缘海拔比南缘高，冬季下雪较厚，因此只有夏季开放，自然环境保持相对原始，没有任何公共交通可以到达。因此南缘成为大多数游客选择游玩的地方，也是我们这次的目的地。

我们是3月11日游玩下羚羊谷之后驾车驶向大峡谷的。先是沿 Navajo Route 20/Coppermine Road 一路向南，拐上 US-89 号公路后接着往南开。在 Cameron 换到 AZ-64 号公路改为向西行驶，最后从大峡谷国家公园的东门进入公园，全程大概140英里，花了两个半小时。进入公园需要每辆车缴费35美元（外国游客还需付额外的100美元），不过因为我事先买了国家公园年票，所以这个费用就免掉了。

沙漠景观（Desert View）

进入公园东门后马上到达第一个观景点，是位于大峡谷东端的“沙漠景观（Desert View）”。我们在停车场停好车后沿着小径往峡谷边缘走，走到观景台之前首先映入眼帘的是一座由红砂砖石筑成的瞭望塔。塔前立的牌子介绍说这座瞭望塔是由设计师 Mary Colter 设计的。她在设计的时候试着把不同形状和图案糅合在一起，门窗的设计也采用了有几百年历史的 T-shaped door 和 Tapered window，外墙也特意使用不平整的砖块，以引入不同阴影的效果。瞭望塔可以排队上去观看风景，不过我们只是在外面照了几张照片就直接去了前面的观景台。

走到观景台的边缘，壮丽的峡谷一下子跳入视野。第一次来大峡谷的我只觉得大地如此辽阔，天空如此深远，沉浸在这壮丽中迷失了自我。

我用目光在峡谷中找到蜿蜒的科罗拉多河，它像一条蓝绿色的缎子，躺在开阔磅礴的峡谷谷底，显得纤弱宁静。可是谁能想到这宽几十公里深一千多米的大峡谷就是这纤弱宁静的缎子在几百万年的时间里刻画出来的呢？

观景台上游客众多，大家都被眼前的壮丽景色震撼，纷纷举起手机和相机记录下这美丽的景色。我边上有一对慈祥的老夫妻，我请老大爷为我们照了几张相。然后我问老大爷需要我为贤伉俪照相吗？老头说 Yes，但是老太太说 No，我的规矩是不跟他照相-:) 我心说，好吧，你们开心就好。开玩笑的，老头和老太太看着挺恩爱的，不知为何老太太不愿意一起照相。话说人和人真的不一样，我在后面碰到另一对老夫妻，老太太可愿意跟老头照相了，反而是老头有点不情愿。

日落

到了大峡谷，当然要去看日落和日出啦。傍晚的时候决定去 Yavapai Point 看日落，我们到达的时候已经夕阳西斜，金黄的阳光洒在东面的峡壁上，和不同石层的峡壁颜色混合起来，呈现出黄、橙、暗红、棕和灰色的层次。再加上天空的金黄和淡蓝，看上去就像一幅油画一样。

渐渐地太阳落到地平线以下。天色没有全黑，反而在峡谷上方泛出一点点紫色。这时候强烈的光线对比消失，更能清楚地看到峡壁上水平分割的不同颜色的岩层。

从 Yavapai Point 往东走，不远处有个地质博物馆，通过模型、图片、视频介绍了大峡谷的人文历史、形成过程、地质和地貌特点。里面有一个地质模型，清楚地显示了一层一层的岩石构造和名称。

博物馆临悬崖的一面是全景玻璃，从里面往外望去也是很震撼的景色。

观星

大峡谷的海拔比较高（大峡谷南缘海拔大概是2100米），而且远离城市的灯光污染，是晚上观星的绝佳地点。晚上观星还是要做一些准备的。首先，晚上的温度很低（大概35华氏度的样子）还有风，需要穿上羽绒服戴上帽子保暖。另外需要准备一个红光手电筒，我们没有买到红光手电筒，就在白光手电筒上蒙了一层红纸对付了。

晚上9点左右出发去观星。刚从租住的小木屋拐上大道就碰到夜出活动的小鹿，于是顺手拍了一张。

观星的地点我们依旧选择了 Yavapai Point。和白天观看日落的人群相比，晚上简直太安静了。除了我们，只有另外一对游客夫妻在欣赏和拍摄星空。我们找了个地方坐下，抬起头欣赏漫天的星星。天上的星星真多啊！我不记得我曾经看过如此多的星星。满天的星星像一颗颗珍珠镶嵌在黑色的玉盘里，有的明亮，有的微弱，有的还在闪啊闪。时不时还能看到闪着灯的飞机在星星下面飞过。

可惜我的天文学知识只局限于北斗七星，只能找到七颗星形成的勺子。另一个冷知识就是勺头和勺柄连接处的那颗星星叫天权，在七星中光线最弱。记得金庸的《射雕英雄传》里提到全真教的北斗七星阵，因为天权的亮度最弱却是连接勺头和勺柄的枢纽，于是由全真七子中武功最强的丘处机担任。

是时候祭出我的索尼单反相机了。这台 Sony A7III 我背了一路没有派上用场，现在该它出马用长曝光模式拍摄星空了。可悲的是我没有带三脚架，只好找了块石头把相机放好，调高ISO，调大光圈，曝光设为30秒，然后听天由命地照了几张。下面这张是照得最清晰的了，逼真地呈现了我们眼睛所见的星空样子。照片里那条橙色的长线是天空中飞过的一架飞机。

2026年拉斯维加斯自驾游（五）：峡谷地国家公园

Posted on 2026-03-20 Edited on 2026-03-22

峡谷地国家公园（Canyonlands National Park）是另一个位于犹他州摩押市的国家公园，距离拱门国家公园很近，所以我们在游玩拱门公园的时候也顺便去了峡谷地公园。

根据维基百科的介绍，峡谷地国家公园是一大片色彩鲜明的大地，被科罗拉多河及其支流侵蚀成数不清的峡谷、台地及孤峰等地形。科罗拉多河与格林河在公园内会合，把公园分成三个区域：天空之岛（Island in the Sky）、针峰区（Needles）和迷宫区（Maze）。我们这次在公园里匆匆而过，只去了天空之岛，所以这篇博文仅记录我们到梅萨拱门观看日出的经历。

梅萨拱门（Mesa Arch）

梅萨拱门恐怕是峡谷地国家公园里最知名的景点。拱门全长8.2米，坐落于海拔约1829米的悬崖边缘。拱门靠悬崖一边正对着东方，早上日出的时候会把拱梁的下方映成橘红色，景色迷人。Windows 7 自带的壁纸里面就有一张梅萨拱门的照片（见下图）。

梅萨拱门的日出景色如此出名，它已经成了全世界摄影爱好者的必到的景点之一。

我们3月10日来到梅萨拱门看日出，那天的日出时间是早上7点38分。为了保证能占到一个拍摄日出的好位子，我们提前了一个半小时大概6点10分到达停车场。一到停车场我们就知道大事不妙，停车场里已经停了十几辆车。等我们抹黑走了10分钟步道到达梅萨拱门的时候，就看到拱门前已经一字排开站满了人，每人前面立一个三脚架，上面架着相机。跟他们交谈得知他们是参加摄影课程的一个小组，老师今天早早带着他们来拍摄著名的梅萨拱门日出。

我不死心，在人墙后面蹭啊蹭，找到一个缝隙就去拍了一张。

然后我们爬上拱门左边的山石，在悬崖边上眺望远方。悬崖边还有其他的游客，静静地享受着日出前的宁静。

时间一分一秒地过去，太阳终于慢慢露出了它的身影。

拱门前的摄影师们也拼命地按着快门，把着一抹阳光和横贯的拱门记录下来。我和朋友不死心，又到人群里面蹭啊蹭，找缝隙拍照片。下面的两张照片第一张是我照的第二章是朋友照的。

我们又欣赏了20分钟美景，然后决定回酒店补觉。在回去的步道上回首再看，朝阳悬在地平线上，把大地染成金黄。梅萨拱门立在那里，游人们依然挤在拱门前，想要把眼前的美景铭刻在脑海里。

在往回开的时候，偶然发现游客中心对面的悬崖有不一样的美景。于是我们下车，拍下了这张如水墨山水画的照片。朝阳藏在云层里，远山近崖被笼罩在雾蒙蒙的光影里，层层叠叠，摄人心魄。

2026年拉斯维加斯自驾游（四）：拱门国家公园

Posted on 2026-03-17 Edited on 2026-03-20

拱门国家公园（Arches National Park）位于犹他州摩押（Moab）市，是世界上最大的自然砂岩拱门集中地，以其独特的地貌以及由红砂岩经过上百万年风蚀形成的各种各样的拱门闻名。园内最小的拱门跨度不到1米，而最大的拱门跨度超过90米。在侵蚀和风化作用下，今天仍有新拱门形成老拱门坍塌。

我们是3月9日来到这里游玩的。进园时每辆车需要支付30美元，不过因为我提前购买了国家公园年票这个费用就免掉了。需要提一句的是，往年旺季的时候（4月份到10月份）需要提前预约才能进入公园，今年由于未知原因全年取消了这个预约机制。不过明年又可能有政策变化，所以2027年以后要去拱门国家公园游玩的朋友还是要到NPS的网站去确认一下。

精致拱门（Delicate Arch）

拱门国家公园里有超过2000座大大小小的拱门，而最著名的莫过于这座精致拱门。精致拱门是公园内最大的自立式拱门，开口有14米高9.8米宽，远看像一个倒立的 U 字谜。精致拱门一开始并不叫这个名字。最初的时候人们称它为“牛仔裤（Cowboy’s Chaps）”或者“盐洗拱门（Salt Wash Arch）”。直到1934年一篇关于拱门国家公园的报道才称之为“精致拱门”，形容它是被匠人精致地凿成的拱门。精致拱门的视觉效果及其震撼又如此有名，以至于犹他州决定将其绘制在汽车车牌上（如下图）。

老话说“无限风光在险峰”，这个正适用于精致拱门。要到达精致拱门需要攀爬一条单程2.4公里的步道。2.4公里的距离不算太长，但是这条步道有146米的高度爬升，地面也不平整，对于我这种平时锻炼少的大叔来说还是很有挑战性的。网上的攻略说精致拱门最美的时候是傍晚日落前后，但是考虑到如果日落前上去，日落后要摸黑下来，难度和危险系数一下子上了几个档次，我们决定一早就去爬这条步道，把最难的事情先做掉。

步道的起点是在 Wolfe Ranch 的停车场，我们早上9点多钟到的时候已经有很多车了。很多游客是一家子一起来的，大人孩子老人一群人准备一起去走这个步道。最神奇的是还有带着很小的baby来的，那种只能用布袋背在身上的小baby，很佩服这样的年轻父母。我们擦好防晒霜，戴上遮阳帽，拿起瓶装水就出发了。先分享一张在接近步道起点的照片，可以看到最上面要爬一块巨大的山石。而且爬到顶之后还有一段距离，从照片的角度根本看不到。

前面比较平缓的部分很快就走完了。但是到了山石需要爬升的那一段我就开始大喘气，走几步喘一下，走几步喘一下。看着身旁的一些年轻人健步如飞超过我，真羡慕他们。咬牙爬到了山石的顶端，前面还有一段路要走，好在剩下的这段路高度爬升不大，不算太难。途中看看路旁的风景，感觉天真蓝啊！

我爬到那个拗口之间让朋友给我拍了一张。人在天地之间显得如此渺小。

接着往上爬。在经过一段靠着悬崖只够两个人错肩而过的小路，一转角眼前豁然开朗。那座著名的精致拱门就优雅地立在那里，拱门的开口里映着蓝天，蓝天下面是静静的雪山，而开口下面一位美女在照相，仿佛点缀在画面上的一朵小花。这个时候阳光是从拱门的后面射过来的，把阴影投在我们这边。可能没有日落时分的辉煌，但是有另一种美。

山上的游客很多，大多数人找个地方坐着，安静地欣赏眼前的美景。我们也找了块岩石坐下，环首四周的风景。雪山、拱门、峡谷、阴影、游客、甚至还有几只飞来飞去的乌鸦，整个画面如此和谐，让人心旷神怡。

精致拱门前有好些游客排队照相。大家都很自觉，轮到的人就到拱门下做各种姿势拍美照，后面的人们则安静地耐心等待。有个小姑娘在拱门下做了大风车，引起一片掌声。而更浪漫的是一位小伙子在拱门下单膝跪地向心爱的女孩成功求婚，引来四周更响的掌声和祝福。而我也未能免俗，去拱门下拍了几张照片。

最后再分享一张精致拱门的照片。拱门右边是个很深的大坑，周围没有护栏，大家去游玩的时候要注意安全。

2026年拉斯维加斯自驾游（三）：羚羊谷及其他

Posted on 2026-03-17 Edited on 2026-03-19

这篇博文记录我们游玩的几个非国家公园但是很有特色的地方。

下羚羊谷（Lower Antelope Canyon）

羚羊谷是位于亚利桑那州佩吉市（Page）纳瓦霍原住民保护区内的狭缝型峡谷，由柔软砂岩经过上百万年的暴洪和风蚀形成，以其独特的地貌和变幻莫测的的光影效果闻名于世。羚羊谷分为上羚羊谷和下羚羊谷。上羚羊谷上窄下宽，内部是A字形状，以夏季中午的光束最为著名。上羚羊谷地面平坦，适合家庭（包括儿童和老人）游玩。下羚羊谷则需要沿金属楼梯向下攀爬到25米深的地底，地形上宽下窄，内部更像V字形状。下羚羊谷地形比上羚羊谷狭窄，变化更多，更具有探险感。上下羚羊谷的游览均需要跟随纳瓦霍族导游游览，而且必须提前几个月预定门票。我们在计划这次行程的时候经过研究决定去下羚羊谷，并提前两个月在网上预订，预定的网站是：Dixie’s Lower Antelope Canyon Tours。根据网上的推荐，我们预定的是上午11点15分的团，因为据说11点前后是光线最好的时候。

下羚羊谷的游览由两家纳瓦霍族旅游公司承包，每天规定只能接待3000人（每一个公司每天只能接待1500人）。游客们被分为15个人一组的游览小组，每组由一位导游带领。我们的导游是一个叫艾萨雅的男孩，看上去像个大学生，一头卷发，声音温柔，整个人给人很清澈的感觉。据他自己介绍，他是Hopi族人，是所有导游中唯一一位非纳瓦霍族人。

在艾萨雅的带领下我们到了峡谷的入口处，在这里等待攀上梯子进入谷底。据导游介绍，在旺季的时候（夏季）游客通常需要等待超过一个小时才能进入谷底。想象一下在超过100华氏度（38摄氏度）高温下无遮挡暴晒等待一个小时的情景，我们很庆幸是三月份来参观的。进入谷底的梯子比较陡，有一段需要面对梯子向下攀爬。下面两张照片是分别从地面和谷底照的入口梯子。

羚羊谷里只允许拍照不允许录像，据说是因为微软和苹果购买了羚羊谷的录像版权。不管怎样，我们一下到谷底就被独特的地貌和奇幻的曲线迷住了。我们随着导游在峡谷里面穿行，一面欣赏着不同光影和颜色的组合，一面惊叹大自然的鬼斧神工，没想到竟然可以看到如此美景。

2026年拉斯维加斯自驾游（二）：生命不息，音乐不止

Posted on 2026-03-17 Edited on 2026-03-18

这次自驾游一个主要的目的就是去看赵传和齐秦在拉斯维加斯的演唱会，这篇博文就先来记录一下这两场演唱会。

赵传

认识赵传还要从我上高中时的某一个冬日说起。当年正放寒假，我顶着漫天的雪花去到家乡唯一的商场闲逛。那时候商场里还有柜台，待售的商品都被放置在柜台里的货架上面，人们购买商品时需要请柜台里的售货员帮忙拿取要买的东西。那时候的音乐歌曲还是以磁带形式传播发行的，而售卖磁带的柜台会有一个大录音机随机播放新到的歌带。就是在这么一个寒冷的冬日，让我撞到了一个高亢的声音唱着《噢！莎莉》。这本应是一首伤心的情歌，可是竟然让这个钢铁般的声音唱出了别样的味道。接着撞入耳膜的便是那首独一无二的《我很丑，可是我很温柔》。一样高亢的声音，把一个故事缓缓道来，让我看到了一个对音乐无限热爱执着追求的灵魂。我瞬间被这个叫赵传的丑男人吸引。

之后的好多年，我听了赵传无数的歌，从《我是一只小小鸟》，到《像我一样骄傲》，到《爱我就给我》，到《爱要怎么说出口》，到《给所有知道我名字的人》，到《男孩看见野玫瑰》，到《因为爱》，到《逆风飞行》，还有很多很多。他的音质独一无二，他的高音清脆有力，他的感情真挚热烈。

赵传演唱会

“我很丑，可是我很温柔。外表冷漠，内心狂热，那就是我~~~” 3月7日的演唱会是以这几句歌词开始的。当歌声响起时，赵传还没有出现在舞台上，可是我仿佛已回到了几十年前的那个冬日，看到那个站在商场柜台边的我，第一次听到这个声音时心灵感受到的碰撞。这一次，我真的可以见到赵传真人了！然后，他出现了，舞台灯光骤亮，赵传就那么站在舞台中间，戴一顶牛仔帽，穿着一件马甲，一条牛仔裤，就那么出现在我眼前。我好激动啊，我的偶像在眼前，一如想象中的丑，可是也一如想象中的铁骨柔情。我的眼睛湿润了，但是不妨碍我跟着唱那些听过无数遍的歌曲。

在演唱会上，赵传和我们分享了他的音乐历程，讲述了他的音乐故事，他成立的红十字乐队以及他和其他音乐人之间的经历。他唱了很多歌迷们耳熟能详的他自己的歌，也唱了一些其他音乐人（包括Beyond乐队和李宗盛）的歌向他们致敬。期间赵传还从舞台上下到观众席和歌迷们握手留影，现场气氛很是热烈。这是一个忘我的夜晚，这是一个欣赏音乐的夜晚，这是一个回忆青春的夜晚。谢谢你，赵传！

2026年拉斯维加斯自驾游（一）：总览

Posted on 2026-03-16 Edited on 2026-03-18

小的时候港台歌曲流行，我听多了就有了几个自己的偶像，其中最喜欢的包括齐秦、童安格、王杰、赵传、周华健等等。今年早些时候得知赵传和齐秦会相继到拉斯维加斯开演唱会，我激动得不得了。从来只是在唱片和电视里听他们唱歌，这次我要到现场去听偶像们唱歌。赵传和齐秦的演唱会一前一后隔开一个星期（赵传是3月7日，齐秦是3月14日），那就借这个机会干脆来个拉斯维加斯周边十日游。

这几篇博文记载了我们这次自驾游的经历。写作它们的目的主要是给自己留个纪念，以后想要回忆的时候可以再翻开来看看。同时也希望能给读到它们的网友一点借鉴，如果我们的游记和经历能够帮助到你们的旅行，哪怕是一点点，我们就很开心了。

行程安排

这次旅行我们计划先到拉斯维加斯听赵传的演唱会，然后在地图上顺时针跑一个大圈，先后游玩拱门国家公园（Arches National Park）、峡谷地国家公园（Canyonlands National Park）、马蹄湾（Horseshoe Bend）、羚羊谷（Antelope Canyon）、大峡谷国家公园（Grand Canyon National Park），最后再回到拉斯维加斯听齐秦的演唱会。根据 Google Map 的估算，这一趟开下来大概有2300英里（3700公里）。下面是我用 Google Map 做的大概行程安排（点击看大图）。

这次自驾游的原则是把行程放得比较宽松，不去追求游玩的国家公园的数目，而是能轻松地把每个地方都玩好，每天开车的时间尽量控制在4.5个小时以内并且尽量不要在天黑时开车。本着这个原则，我们把原本计划要去的锡安国家公园和布莱斯峡谷国家公园从行程上去掉了。最终的行程如下：

  • 3月6日从家出发开到Bakersfield，车程约4.5个小时。在Bakersfield住一晚上。
  • 3月7日从Bakersfield出发开往拉斯维加斯，车程约4个小时。晚上听赵传演唱会。在拉斯维加斯住一晚上。
  • 3月8日从拉斯维加斯出发开往Moab UT，车程月6.5个小时。傍晚到峡谷地国家公园看日落。在Moab住第一个晚上。
  • 3月9日白天游玩拱门国家公园，晚上在拱门国家公园观星。在Moab住第二个晚上。
  • 3月10日早上到著名的峡谷地国家公园里的Mesa Arch看日出。然后回酒店补觉。中午从Moab UT开往Page AZ，车程约4.5个小时。途经163号公路上的Forest Gump Point（这是电影《阿甘正传》里面一个经典画面的拍摄地点）。傍晚到马蹄湾看日落。在Page住一晚上。
  • 3月11日早上睡到自然醒，然后打卡网红点Shell加油站，接着游玩下羚羊谷。下午开往大峡谷国家公园，车程约2.5个小时。傍晚在Yavapai Point观看日落，晚上在Yavapai Point观星。在大峡谷住第一个晚上。
  • 3月12日早上在Mather Point看日出，下午坐公园里的红线接驳车游玩Hermit Road，傍晚在大峡谷边上的Rim Trail散步。在大峡谷住第二个晚上。
  • 3月13日早上从大峡谷开往拉斯维加斯，车程约4.5个小时。中间在Kingman AZ停留参观66号公路博物馆。晚上在拉斯维加斯大道上逛街。在拉斯维加斯住第一个晚上。
  • 3月14日白天在Las Vegas North Premium Outlets逛街购物，晚上看齐秦演唱会。在拉斯维加斯住第二个晚上。
  • 3月15日从拉斯维加斯开回家，车程约8.5个小时。

这个行程里3月8日和3月15日两天需要开车超过4.5个小时，这个是赶路需要，而且我们有两个人可以换手轮流开车，最后开下来觉得还可以接受，不是特别累。

阅读完陈浩基的《13.67》

Posted on 2026-02-23

今年开年许的愿望之一就是重新开始读书。目标很低：一个月读一本。一月份开始读 Ursula Le Cuin 的《No Time to Spare》，前面几章关于年长之后的人生思考以及她家猫猫的内容我读得还比较有兴趣。不过后面开始写她当作家时和读者的一些交流互动不是很吸引我，就暂时放弃了。

之后选择了香港作家陈浩基的《13.67》。这是一本中篇推理小说，全书有六个本格推理故事，作家用年代倒叙的手法，通过对关振铎和骆小明警察师徒破获的六个刑事大案的描写，描绘了1967年至2013年香港风起云涌的社会变迁（后来才明白书名是从这两个年代取的）。里面的故事构思巧妙，很多反转，首尾衔接，是不可多得的犯罪小说佳品。推荐给大家看。

How to Sideload Kodi to Apple TV Using atvloadly

Posted on 2026-02-15 Edited on 2026-02-23

Overview

The goal of this article is to share how I installed (sideloaded) Kodi to my Apple TV 4K (2nd Gen). Kodi is a media center and entertainment hub that I have been using since pre-Kodi era (when Kodi was XBMC). It has a beautiful interface and is easy to use. I mainly use Kodi to watch and record over-the-air (OTA) TV programs, as well as watch ripped blueray movies. Kodi has powerful add-ons that can expand and enhance its features and capabilities. It officially supports Windows, linux, and android. Unfortunately, Apple TV, which runs tvOS, does not have an official version of Kodi in App Store. This means we cannot download and install Kodi from App Store. Instead, we need to find some unofficial way to install or sideload it to Apple TVs.

atvloadly is a web service that supports sideloading apps on Apple TV. In this article, I’ll share how I setup an atvloadly docker and how I used atvloadly to sideload Kodi to my Apple TV. Hopefully this will help someone else who’s looking to do the same.

Workflow and Prerequisites

  1. Workflow

    • Setup an atvloadly docker.
    • Add an Apple account to atvloadly.
    • Pair the atvloadly web service with the Apple TV.
    • Sideload Kodi to the Apple TV.

  2. Prerequisites

    • A computer running linux. Unfortunately, this method does not work on Windows. It doesn’t need to be a full-fledged laptop or desktop. You can use a Raspberry Pi if you have one lying around. I used a single board computer called Rock64 running a stripped down version of Debian.
    • A burned Apple account. As noted on the atvloadly website, it is a good ideal to use a dedicated Apple ID or Apple account for sideloading. A free Apple account will work. I just created a new free Apple account for this purpose. You can create one on the Apple Developer Account webpage.
    • A phone for two-factor authorization (2FA). When creating a new Apple account for sideloading, you need a phone for 2FA. atvloadly needs to be authorized as a trusted device (it will be virtualized as a MacBook). When adding the Apple account to atvloadly, Apple will send a 2FA verification code to the registered phone number and you will need to enter the code and authorize it.
